## ✨ Looker Audit Skill - Key Features

An AI-powered assistant designed to automate Looker instance health checks, performance auditing, and LookML best practice validation.

- **Performance Auditing**: Identifies slow Explores, heavy dashboards, and scheduling hotspots.
- **Error Detection**: Spots failed PDTs, broken datagroups, and query errors.
- **Best Practice Validation**: Checks for missing primary keys, excessive dynamic fields, and anti-patterns.
- **Bespoke Reporting**: Generates actionable reports with direct links to violations and code snippets.
